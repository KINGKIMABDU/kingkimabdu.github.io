# KINGKIMABDU Portfolio Website

Welcome to the official repository for my personal website, **KINGKIMABDU // Crafted to Create**.

This website serves as my digital hub, showcasing my projects, interests, and professional persona as a tech enthusiast, inventor, and student.

## 🚀 Live Website
You can view the live site here: **[https://kingkimabdu.github.io/](https://kingkimabdu.github.io/)**

## 🛠️ Technology Stack
* **HTML5**: Structured semantic markup.
* **CSS3**: Custom styling using CSS variables for a themeable design, Flexbox for layout, and media queries for responsiveness.
* **JavaScript (ES6)**:
    * Dynamic typewriter effect for the hero section.
    * Auto-updating copyright year.

## 🎨 Design Features
* **Minimalist Aesthetic**: Clean dark theme with accent colors (`#5f643d`, `#a2a569`).
* **Responsive Design**: Optimized for mobile, tablet, and desktop views.
* **Interactive Elements**:
    * Hover animations on images and social icons.
    * CSS-based tooltips for social platforms.
    * Typing animation cursor.
* **Typography**: Uses the 'Sen' font family from Google Fonts for a modern feel.

## 🧩 SEO & Structured Data
The website includes embedded JSON-LD structured data to help search engines understand the content and persona, enhancing visibility for searches related to tech, bodybuilding, and invention.
